# mock interview feedback project
A web-based mock interview feedback system that helps users practice interviews and receive performance feedback.
## 💻 Technologies Used
- Python
- Flask
- HTML
- CSS

## ⚙️ Features
- User login interface
- Interview questions practice
- Feedback system
- performance evaluation
- Summary page

## ▶️ How to Run
1. Install Python on your system
2. Install Flask (pip install flask)
3. Run app.py
4. Open browser and go to localhost
